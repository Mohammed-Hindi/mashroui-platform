# Mashroui Platform

## Graduation Project Proposal

Mashroui is a proposed digital platform for managing and automating academic graduation projects.

The platform aims to transform traditional graduation project management into a unified digital environment that connects students, supervisors, examiners, and academic administration within a single system.

### Main Idea

* Project proposal management.
* Student team management.
* Project progress tracking.
* Supervisor-student communication.
* Smart discussion scheduling.
* Academic proposal review.
* Digital grading and evaluation.
* Final project archiving.

### Technologies

* Laravel 12
* Vue.js 3
* Inertia.js
* MySQL
* Chart.js

### Status

This repository currently contains the project proposal and interface prototype prepared for supervisor review and discussion.
